# Security And Privacy

The Self Publishing AI Playbook is a public Markdown skill package. It should not contain secrets, credentials, or private data.

## Do Not Commit

- API keys, tokens, OAuth credentials, or passwords
- `.env` files or local credential files
- Customer data, email lists, or launch team contact lists
- Private client examples
- Private Gen AI University system details
- Links to private repositories or internal tools

## Reporting A Problem

If you find private data, credentials, or a security issue in this repository, do not open a public issue with the sensitive details.

Contact Gen AI University through [genaiuniversity.com](https://genaiuniversity.com) and include:

- The affected file path
- A short description of the issue
- Whether the issue is currently public

## Public Package Boundary

This package should draft, organize, and guide. It should not assume access to private connectors, inboxes, social schedulers, CRMs, payment accounts, customer databases, or any other private system.
