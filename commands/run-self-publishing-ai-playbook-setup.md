# Run Self Publishing AI Playbook Setup

Use this slash command as the entry point into the full setup skill.

Run the complete onboarding process in `skills/self-publishing-ai-playbook-setup/SKILL.md`.

The setup must:

1. Explain that the playbook acts as an AI publishing assistant, not a replacement author or done-for-you ghostwriter.
2. Ask where the author is right now in the book journey.
3. Collect the core book, author, audience, timeline, and platform context.
4. Create a copy/paste project context block.
5. Recommend the next specialist skill based on the author's current state.
6. Offer to run that recommended skill immediately.

If the author already has context, use it and skip any questions that are already answered.
